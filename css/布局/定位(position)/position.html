static 是默认值 position:static 元素不会被特殊的定位
一个static元素标识他不会被 positioned 一个position属性被设置为其他值的元素则标识他会被 positioned
relative 表现的和 static 一样，除非你添加了一些额外的属性。
在一个相对定位（position属性的值为relative）的元素上设置 top 、 right 、 bottom 和 left 属性会使其偏离其正常位置。 其他的元素的位置则不会 因为该元素发生位置改变的影响来弥补它偏离后剩下的空隙。
