# 前端简介
--------

## 前端的定义
    1. 切图   静态页面
    2. 美工
    3. 前端工程师(负责界面的交互实现，界面处理，界面适配）
##  后端工程师的定义
    1. 负责接口的实现
    2. 性能   （中等企业，数据量一般都在百万级别） 
    3. 并发量的处理（ 同时访问的请求量 ）
## 前端工程师 要做什么
    1.  对与小企业（一般没有前后端分离，干活讲究的是 猛，糙，块） 啥都做
    2.  对与中大型企业
        2.1 对与前后端分离（重视前端交互） 写页面（html+css) 写交互(javaScript)
        2.2 前后端不分离（更重视后端）    写 freemarker || jsp || servlet 【需要在2.1基础上额外的学习模板语法】
